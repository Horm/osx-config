# OSX Configuration
My configuration and apps for OSX

## Configuration

### Folders

In user directory (e.g. `horm`) create a folder `Development` with these subfolders: 

- Web
- Mobile
- HW
- Other

Then add/orginize folders in Finder side menu (user directory, Development, Google Drive, etc.)

### Other

- Silent Touchpad
- Biggest Resolution
